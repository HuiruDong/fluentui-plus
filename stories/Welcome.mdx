# 欢迎使用 Fluent UI Plus

Fluent UI Plus 是一套专为中国企业级应用设计的 React 组件库，基于 Microsoft Fluent UI 设计体系，结合中国用户的使用习惯和场景需求。

## ✨ 特性

- 🎨 **现代设计语言** - 基于 Fluent Design System 设计理念
- 🌏 **国际化支持** - 内置中文支持，符合中国用户习惯
- 📱 **响应式设计** - 适配各种屏幕尺寸和设备
- ⚡ **高性能** - 基于现代化技术栈，性能优越
- 🛡️ **TypeScript** - 完整的 TypeScript 类型定义
- 🎯 **企业级** - 适合企业级应用开发需求

## 🚀 快速开始

### 安装

```bash
npm install @luoluoyu/fluentui-plus
# 或
yarn add @luoluoyu/fluentui-plus
# 或
pnpm add @luoluoyu/fluentui-plus
```

### 使用

```tsx
import React from 'react';
import { Button, Tag } from '@luoluoyu/fluentui-plus';

function App() {
  return (
    <div>
      <Button variant='primary'>主要按钮</Button>
      <Tag>默认标签</Tag>
    </div>
  );
}

export default App;
```

- **可扩展性** - 灵活的组件扩展能力
